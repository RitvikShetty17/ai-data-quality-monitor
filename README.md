# AI-Powered Data Quality Monitor

An automated data quality monitoring system that profiles datasets, validates business rules using Great Expectations, and uses the Claude API to generate plain-English explanations of data anomalies — with Slack alerting and a live Streamlit dashboard.

## Tech Stack
- **Python** — core profiling and orchestration
- **Great Expectations** — automated data validation rules
- **Claude API (Anthropic)** — AI-generated anomaly explanations
- **Streamlit** — live data health dashboard
- **Slack Webhooks** — real-time alerting
